Configuração do GitHub
Como configurar
1
Crie um repositório público no GitHub chamado pontohora e ative o GitHub Pages (Settings → Pages → Branch: main)
2
Clique em "Gerar token" acima. Marque o escopo repo e crie o token.
3
Preencha os campos acima com seu usuário, nome do repositório e o token gerado.
4
Clique em Salvar e conectar. Os arquivos usuarios.json e registros.json serão criados automaticamente no repositório.
