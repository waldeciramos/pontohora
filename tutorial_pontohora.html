Etapa 1 — Criar o repositório no GitHub

1
Criando o repositório
Acesse github.com, clique em + → New repository e preencha as informações abaixo.
  • Nome: pontohora
  • Visibilidade: Público
  • Descrição: Sistema de registro de ponto hospedado em GitHub Pages
  • Ative o toggle "Adicionar arquivo README"
  • Clique em Criar repositório
2
Subindo o index.html
Com o repositório criado, faça upload do arquivo index.html do PontoHora.
  • Clique em "Adicionar arquivo" → "Carregar arquivos"
  • Arraste o arquivo index.html
  • Clique em Confirmar alterações
Repositório criado com README.md e index.html — tudo certo!
3
Ativar o GitHub Pages
Para o site ficar acessível publicamente na internet.
  • Clique na aba Configurações do repositório
  • No menu lateral, clique em Pages
  • Em Branch, selecione principal e pasta / (root)
  • Clique em Save
  • Aguarde ~1 minuto — o site estará em waldeciramos.github.io/pontohora/

Etapa 2 — Gerar o Personal Access Token

4
Criar o token de acesso
O token permite que o PontoHora grave dados diretamente nos arquivos JSON do repositório.
  • Clique na sua foto → Configurações
  • Role até o final → Configurações do desenvolvedor
  • Clique em Tokens de acesso pessoal → Fichas (clássicas)
  • Clique em Gerar novo token (classic)
  • Observação: PontoHora
  • Validade: No expiration
  • Escopos: marque repositório (o primeiro checkbox principal)
  • Clique em Gerar token
5
Copiar o token gerado
O token aparece apenas uma vez — copie imediatamente e guarde em local seguro.
O token começa com ghp_ e só aparece nesta tela. Após sair da página não é possível ver novamente — guarde no bloco de notas!

Etapa 3 — Configurar e acessar o PontoHora

6
Conectar ao repositório
Abra o site e preencha as informações de conexão com o GitHub.
  • Acesse waldeciramos.github.io/pontohora/
  • Usuário GitHub: waldeciramos
  • Nome do repositório: pontohora
  • Token: cole o token copiado (ghp_...)
  • Clique em Salvar e conectar
Os arquivos usuarios.json e registros.json são criados automaticamente no repositório!
7
Criar sua conta e acessar
Após conectar, você será redirecionado para a tela de login. Crie sua conta para começar.
  • Clique em Criar conta
  • Preencha nome, CPF, cargo, departamento, e-mail e senha
  • O primeiro usuário cadastrado vira admin automaticamente
  • Faça login com o e-mail e senha cadastrados
  • Bata o ponto! Cada batida gera um commit no GitHub 🎉

🎉 PontoHora no ar!

Os dados ficam salvos nos arquivos JSON do próprio repositório.
